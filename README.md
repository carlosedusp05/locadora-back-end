# Projeto: Sistema de Locadora (Simulação Didática)
Este projeto foi desenvolvido com fins didáticos na escola **SENAI**, com o objetivo de simular o funcionamento de uma locadora através de um sistema de gerenciamento simples.

A aplicação permite realizar operações básicas relacionadas ao gerenciamento de filmes, atores, diretores, generos entre outros utilizando tecnologias modernas utilizadas no mercado de desenvolvimento.

O projeto também tem como finalidade praticar conceitos de:
- Desenvolvimento **Back-end**
- **Banco de dados relacional**
- **APIs REST**
- **Versionamento de código**
- **Testes de requisição**

  
## Tecnologias Utilizadas

As seguintes ferramentas e tecnologias foram utilizadas no desenvolvimento do projeto:
- **Node.js** → Ambiente de execução JavaScript no servidor  
- **Prisma ORM** → Mapeamento objeto-relacional para comunicação com o banco de dados  
- **MySQL / SQL (MySQL Workbench)** → Gerenciamento do banco de dados  
- **Postman** → Teste das rotas da API  
- **Git** → Controle de versão do projeto  
- **GitHub** → Hospedagem e gerenciamento do repositório
